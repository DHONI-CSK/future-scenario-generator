# 🚀 Future Scenario Generator

An AI-powered web application that generates possible future scenarios based on current signals and trends.

---

## 📌 Project Overview

The Future Scenario Generator is designed to help users analyze how present-day factors like technology, climate change, and social trends can shape the future.

Users can input multiple signals and generate different possible outcomes such as optimistic, pessimistic, and alternative scenarios.

---

## ✨ Features

- 📊 Add multiple future signals
- ⚡ Generate different future scenarios
- 🔍 Compare scenarios side-by-side
- 📈 Visual representation of outcomes
- 🧠 AI-based logic for scenario generation
- 🎯 Interactive and user-friendly interface

---

## 🛠️ Technologies Used

- HTML, CSS, JavaScript
- React (Vite)
- AI logic (Kiro AI generated)
- GitHub for version control

---

## ⚙️ How It Works

1. User enters signals (e.g., AI growth, climate change)
2. System processes inputs
3. Generates multiple future scenarios:
   - Optimistic
   - Pessimistic
   - Balanced
   - Wildcard
4. Users can compare scenarios and analyze outcomes
